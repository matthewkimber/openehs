HOW TO SETUP THE DYMO 450 Printer

1.Do not plug the printer in yet!

2.Run the program titled: DLS8Setup.8.2.3.1026-BETA.  You MUST have the DLS software installed and it must be atleast version 8.2.3.1026.  If it is not atleast this version, the printer will not workgin OpenEHS.

3.Follow the instructions (just click next over and over)

4.THe last step will tell you to plug in the Dymo printer. Plug in the printer.

5.Restart the computer.  You are not prompted to restart the computer, but in order for the printer to work with OpenEHS, you must restart.  

6.Once the computer is restarted, the printer should work with OpenEHS.
