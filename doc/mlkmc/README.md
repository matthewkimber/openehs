# MLKMC EHS — Documentation

Project documentation for the Martin Luther King Memorial Clinic (MLKMC) electronic healthcare system, the project from which OpenEHS originated.

## Contents

- **Planning & management** — Project Charter, Project Proposal, Project schedule (`.mpp`), Plan of work (SDD/SRSS), and Responsibility Breakdown.
- **Specification & design** — Specification and Design, Design, Use Cases, and SSRS documents.
- **`pdf/`** — PDF exports of the charter, proposal, design, specification, and SSRS documents.
- **`examples/`** — sample reference material (e.g. medical examples).

> Source documents are Word (`.docx`), Excel (`.xlsx`), and MS Project (`.mpp`) files; see `pdf/` for readily viewable exports.
