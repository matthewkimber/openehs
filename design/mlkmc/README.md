# MLKMC EHS — Design

Design artifacts for the Martin Luther King Memorial Clinic (MLKMC) electronic healthcare system, the project from which OpenEHS originated.

## Contents

- **`MLKMC EHS - ERD.vsd`** — entity-relationship diagram.
- **`MLKMC EHS - Conceptual Database.vsd`** — conceptual database model.
- **`uml/`** — UML diagrams (use cases, classes, sequence, activity, state machines, architecture, package diagram).
- **`mockups/`** — UI mockups (Balsamiq `.bmml` files) for the admin, doctor, and reporting screens, plus supporting assets.

> Most files are Visio (`.vsd`) and Balsamiq (`.bmml`) documents and require the corresponding tools to view.
